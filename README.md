# 🌐 Personal Portfolio
A high-fidelity, brutalist engineering portfolio designed for the Python 3.14 era. This project is a technical manifesto and showcase of low-level infrastructure work, AI integration, and open-source contributions.

### 🔗 [Live Deployment ↗](https://raushan-portfolio.onrender.com)

---

## 🛠️ Technical Stack & Architecture

- **High-Performance ASCII Engine**: A custom DOM-based particle system that handles real-time coordinate geometry calculations for interactive ASCII art dispersal.
- **Brutalist Typography Logic**: Leverages `Space Grotesk` and `JetBrains Mono` with high-contrast `-webkit-text-stroke` effects for a premium agency aesthetic.
- **Vanilla Performance**: Zero-dependency frontend architecture built with **Tailwind CSS** and **Vanilla JS** to ensure sub-100ms LCP (Largest Contentful Paint) times.
- **Dynamic Scroll States**: Implemented using `IntersectionObserver` to trigger staggered reveal animations across the vertical and horizontal axes.
- **Cross-OS Asset Pipeline**: Developed on **Arch Linux**, utilizing a dual-boot workflow to migrate legacy assets from Windows NTFS partitions.

## 🧬 Core Focus Areas

- **Open Source**: Active contributor to the **Wikimedia Foundation** (Pywikibot) and **The Linux Foundation**, focusing on test suite optimization and legacy script modernization.
- **AI/ML Infrastructure**: Developing the **S.I.X Core**—a voice-to-text and lyrics engine utilizing OpenAI Whisper and HuggingFace Transformers.
- **Backend Scaling**: Researching knowledge graph automation and high-concurrency Python architectures (3.14+).
